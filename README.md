## Como rodar o projeto?
Primeiro instalamos nossas dependências
```bash
npm install
```
... depois precisamos iniciar o json-server
```bash
npm run server
```
... e rodamos nossa aplicação em uma nova instância do terminal.
```bash
npm start
```
